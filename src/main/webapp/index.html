Simple Web Application 101

Creating Person

Use the following template to create people in the database. 
http://localhost:8080/Module8/webservice.jsp?create&firstName=FIRSTNAME&lastName=LASTNAME&employeeNumber=EMPLOYEENUMBER&email=EMAIL@fit.edu
Example

Updating Person

Use the following template to update people in the database. 
http://localhost:8080/Module8/webservice.jsp?update&id=IDNUMBER&firstName=FIRSTNAME&lastName=LASTNAME&employeeNumber=EMPLOYEENUMBER&email=EMAIL@fit.edu
Example: 
http://localhost:8080/Module8/webservice.jsp?update&id=10&firstName=FIRSTNAME&lastName=LASTNAME&employeeNumber=EMPLOYEENUMBER&email=EMAIL@fit.edu

Listing All Person

Use the following template to update people in the database. 
http://localhost:8080/Module8/webservice.jsp?list
Example
